web

localStorage

  • 로컬에서 도메인 별로 지속되는 storage 공간(저장하는 공간)
  • 제한 시간이 없고 브라우저를 닫기 전까지는 사라지지 않는다
  • 값을 지우려면 직접 지워야 한다
  • 여러 새 창이나 새 탭이 열려도 데이터 공유가 가능하다

sessionStorage

  • 세션 (프로세스, 탭, 브라우저)이 종료될 때까지 지속되는 storage
  • 새 창이나 새 탭이여도 데이터 공유가 되지 않음

프로퍼티 타입으로 key와 value가 있음. 자바스크립트에서 object(객체)

저장할 때 무조건 문자열만 저장할 수 있다

값 저장, 불러오기

setItem('key', 'value'): 저장

getItem('key'): 불러오기

removeItem('key'): 삭제. 해당 키를 받으면 한 쌍을 동시에 제거

clear(): 모든 데이터 지우기

length: 모든 key, value 해당하는 길이 반환

Object.keys(): 현재 웹 스토리지에 있는 모든 key만 가져온다

웹 스토리지 확인 방법

크롬 > F12 > application 탭에 가면 좌측에서 확인할 수 있다
